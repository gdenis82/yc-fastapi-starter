# FastAPI Kubernetes Project

Проект организован с соблюдением ключевых принципов разработки ПО для обеспечения масштабируемости, читаемости и легкости поддержки.

## Архитектура проекта

```text
.
├── app/                  # Основной код приложения
│   ├── api/              # Маршрутизация и обработчики (Endpoints)
│   │   ├── endpoints/    # Конкретные группы маршрутов
│   │   └── api.py        # Агрегация всех роутеров
│   ├── core/             # Ядро приложения (конфигурация, логирование)
│   └── main.py           # Точка входа и фабрика приложения (FastAPI)
├── k8s/                  # Манифесты Kubernetes
├── terraform/            # Инфраструктура как код (Yandex Cloud)
├── Dockerfile            # Инструкция для сборки контейнера
├── deploy.ps1            # Скрипт автоматизированного деплоя
└── DEPLOY.md             # Подробная инструкция по развертыванию
```

## Развертывание
Подробная инструкция находится в файле [DEPLOY.md](DEPLOY.md).
